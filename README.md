PIPELINE DE DADOS - FORMULA 1
Este projeto consiste na criação de uma pipeline de dados completa utilizando a plataforma Databricks, simulando um ambiente real de Engenharia de Dados com foco em performance e organização.
🛠️ Tecnologias Utilizadas
• Linguagens: SQL (SparkSQL / PL/SQL context) e Python (PySpark)
• Plataforma: Databricks
• Armazenamento: Delta Lake (Arquitetura Medalhão)
• Banco de Dados: Manipulação de tabelas relacionais e queries complexas.
